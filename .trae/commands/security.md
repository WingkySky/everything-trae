# 安全审计 (Security Audit)

## 命令
`/security`

## 功能描述
对代码进行全面的安全审计，发现潜在的安全漏洞和风险。

## 使用场景
- 安全审查
- 发布前检查
- 定期安全扫描
- 合规性检查

## 审计内容
1. **注入风险**
   - SQL 注入
   - XSS 攻击
   - 命令注入
   - 模板注入

2. **认证授权**
   - 会话管理
   - 权限控制
   - 认证流程
   - 密码安全

3. **数据安全**
   - 敏感信息加密
   - 数据验证
   - 输入过滤
   - 日志脱敏

4. **API 安全**
   - 速率限制
   - 输入验证
   - 错误处理
   - 日志记录

5. **依赖安全**
   - 依赖漏洞
   - 版本更新
   - 许可证合规
   - 供应链安全

## 输出格式
- 安全问题列表
- 按严重程度分类
- 具体的修复建议
- 安全评分

## 示例
```
用户: /security
AI: [生成详细的安全审计报告]
```

## 注意事项
- 全面覆盖安全方面
- 提供具体的修复方案
- 考虑业务场景
- 遵循安全最佳实践
