# 代码审查 (Code Review)

## 命令
`/review`

## 功能描述
启动代码审查模式，让 AI 对代码进行全面的审查和分析。

## 使用场景
- 提交代码前
- 代码重构后
- Pull Request 评审
- 安全审计

## 审查内容
1. **代码质量**
   - 代码规范
   - 复杂度
   - 可读性

2. **安全性**
   - 潜在漏洞
   - 输入验证
   - 权限控制

3. **性能**
   - 性能瓶颈
   - 优化建议
   - 资源使用

4. **测试覆盖**
   - 测试完整性
   - 覆盖率
   - 边界条件

## 输出格式
- Markdown 格式
- 按严重程度分类问题
- 提供具体修复建议
- 总结整体质量评分

## 示例
```
用户: /review
AI: [生成详细的代码审查报告]
```

## 注意事项
- 专注于代码本身
- 提供建设性反馈
- 遵循项目规范
