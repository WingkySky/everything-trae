# 规划任务 (Plan Task)

## 命令
`/plan`

## 功能描述
启动规划模式，让 AI 先分析需求并生成详细的开发计划，然后再开始编写代码。

## 使用场景
- 新功能开发
- 大型重构
- 项目初始化
- 架构设计

## 工作流程
1. 用户输入需求描述
2. AI 分析需求并生成详细计划
3. 用户审查和批准计划
4. AI 根据批准的计划执行开发

## 输出格式
- Markdown 格式
- 包含任务列表
- 标注技术选型
- 提供风险评估

## 示例
```
用户: /plan 开发一个用户管理系统
AI: [生成详细的开发计划]
```

## 注意事项
- 规划阶段不要编写代码
- 计划要详细且可执行
- 包含所有必要的步骤
