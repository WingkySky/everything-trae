# 代码优化 (Code Optimization)

## 命令
`/optimize`

## 功能描述
对代码进行全面的优化，包括性能优化、结构优化和最佳实践应用。

## 使用场景
- 性能瓶颈排查
- 代码重构
- 技术债务清理
- 最佳实践应用

## 优化类型
1. **性能优化**
   - 算法优化
   - 数据库查询优化
   - 缓存策略
   - 并发处理

2. **结构优化**
   - 代码重构
   - 模块化改进
   - 依赖管理
   - 命名规范

3. **安全优化**
   - 漏洞修复
   - 输入验证
   - 权限控制
   - 日志记录

4. **可维护性优化**
   - 代码可读性
   - 注释完善
   - 错误处理
   - 测试覆盖

## 输出格式
- 优化建议列表
- 具体的代码示例
- 性能对比数据
- 风险评估

## 示例
```
用户: /optimize
AI: [生成详细的优化建议]
```
