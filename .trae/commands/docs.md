# 文档生成 (Documentation Generation)

## 命令
`/docs`

## 功能描述
为代码生成完整的文档，包括 API 文档、使用示例和最佳实践。

## 使用场景
- 新功能开发
- API 发布
- 项目文档更新
- 技术分享准备

## 文档类型
1. **API 文档**
   - 端点说明
   - 请求/响应格式
   - 错误码说明
   - 使用示例

2. **开发文档**
   - 环境搭建
   - 项目结构说明
   - 开发规范
   - 常见问题

3. **用户文档**
   - 功能介绍
   - 使用指南
   - 最佳实践
   - 故障排查

4. **架构文档**
   - 系统架构图
   - 模块关系说明
   - 技术选型理由
   - 部署方案

## 输出格式
- Markdown 格式
- 清晰的目录结构
- 完整的示例代码
- 常见问题解答

## 示例
```
用户: /docs src/api/
AI: [生成完整的 API 文档]
```

## 注意事项
- 文档要准确完整
- 示例代码可运行
- 包含常见问题
- 定期更新维护
