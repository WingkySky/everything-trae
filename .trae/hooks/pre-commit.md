# 预提交钩子 (Pre-commit Hook)

## 触发时机
在执行 `git commit` 时自动触发

## 执行任务
1. 代码格式检查
   - ESLint 检查
   - Prettier 格式化
   - TypeScript 类型检查

2. 测试验证
   - 运行单元测试
   - 检查测试覆盖率

3. 代码审查
   - 检查代码规范
   - 检查注释完整性

4. 安全扫描
   - 检查敏感信息
   - 检查依赖漏洞

## 失败处理
- 任何检查失败都会阻止提交
- 提供详细的错误信息
- 建议修复方案

## 跳过钩子
- 使用 `--no-verify` 参数（不推荐）
- 仅在紧急情况下使用
