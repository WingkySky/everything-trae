# 前端架构设计技能

## 触发场景
当用户需要设计前端架构、创建新项目或重构前端代码时，自动触发前端架构设计技能。

## 架构原则

### 组件设计
- 使用原子设计原则
- 组件职责单一
- 组件可复用
- 组件易于测试

### 状态管理
- 选择合适的状态管理方案
- 明确状态所有权
- 避免状态冗余
- 实现状态持久化

### 路由配置
- 使用嵌套路由
- 实现懒加载
- 配置路由守卫
- 处理路由参数

### 样式方案
- 使用 CSS Modules 或 CSS-in-JS
- 统一设计系统
- 实现主题切换
- 支持响应式设计

## 输出规范
- 架构图（如 C4 模型）
- 技术选型说明
- 目录结构建议
- 最佳实践指南
