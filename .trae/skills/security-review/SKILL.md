# 安全审查技能

## 触发场景
当用户需要审查代码安全性、进行安全审计、发现漏洞或实现安全功能时，自动触发安全审查技能。

## 安全审查清单

### 身份验证与授权
- 实现强密码策略
- 使用多因素认证（MFA）
- 实现基于角色的访问控制（RBAC）
- 令牌过期和刷新机制

### 输入验证
- 验证所有用户输入
- 防止 SQL 注入
- 防止 XSS 攻击
- 防止 CSRF 攻击
- 文件上传安全检查

### 数据安全
- 敏感数据加密存储
- 传输层加密（TLS/SSL）
- 密钥安全存储
- 数据脱敏处理

### API 安全
- 实现速率限制
- 使用 HTTPS
- 验证请求来源
- 安全的 API 响应格式

### 错误处理
- 避免泄露敏感信息
- 统一错误响应格式
- 记录安全事件日志

## 安全编码规范

### 常见漏洞防护
- 输入验证与过滤
- 输出编码
- 参数化查询
- 安全配置

### 依赖项安全
- 定期更新依赖
- 使用安全扫描工具
- 审查第三方库

## 输出规范
- 安全审计报告
- 漏洞修复建议
- 安全配置清单
- 安全测试用例

## 使用示例
- "审查这段代码的安全性"
- "进行安全漏洞扫描"
- "实现用户认证功能"
- "修复 SQL 注入漏洞"
