# API 设计技能

## 触发场景
当用户需要设计 API、创建 RESTful 服务或 GraphQL 服务时，自动触发 API 设计技能。

## 设计原则

### RESTful 设计
- 使用名词复数表示资源
- 使用适当的 HTTP 方法
- 实现版本控制
- 统一错误处理

### GraphQL 设计
- 设计合理的类型系统
- 实现查询优化
- 处理 N+1 问题
- 实现缓存策略

### API 文档
- 使用 OpenAPI 规范
- 提供完整的示例
- 包含错误处理
- 维护版本历史

### 安全设计
- 实施认证授权
- 输入验证
- 速率限制
- 日志记录

## 输出规范
- API 设计文档
- OpenAPI 规范文件
- 示例请求和响应
- 安全策略说明
