# 研究优先技能

## 触发场景
当用户需要实现新功能、解决复杂问题或学习新技术时，自动触发研究优先技能。强调在动手编码前先进行充分的技术调研。

## 研究流程

### 1. 问题分析
- 明确需求和目标
- 识别技术挑战
- 确定评估标准

### 2. 技术调研
- 搜索官方文档
- 查找最佳实践
- 分析社区解决方案
- 评估工具和库

### 3. 方案评估
- 对比多种方案
- 考虑性能、可维护性
- 评估学习成本
- 权衡利弊得失

### 4. 决策记录
- 记录选型理由
- 保存参考资料
- 总结经验教训

## 研究方法

### 信息源优先级
1. 官方文档和规范
2. 权威技术博客
3. 开源项目源码
4. 社区讨论和问答

### 工具使用
- 网络搜索获取最新信息
- 代码搜索学习最佳实践
- 技术文档深度阅读

## 输出规范
- 技术调研报告
- 方案对比分析
- 参考资料清单
- 实施建议

## 使用示例
- "研究如何实现实时协作功能"
- "调研 React 状态管理方案"
- "分析这个问题的最佳解决方案"
- "学习如何使用这个新框架"
