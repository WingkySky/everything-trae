# 安全规则

## 输入验证
- 所有用户输入必须验证
- 使用白名单验证
- 限制输入长度
- 过滤特殊字符

## 认证授权
- 使用 JWT 进行会话管理
- 实施适当的权限控制
- 定期更新会话令牌
- 实现登录失败限制

## 数据安全
- 敏感数据必须加密
- 不要在代码中硬编码密钥
- 使用环境变量管理配置
- 实施数据访问控制

## API 安全
- 实施速率限制
- 验证所有 API 请求
- 使用 HTTPS
- 实施适当的 CORS 策略

## 日志安全
- 不要记录敏感信息
- 实施适当的日志级别
- 实现日志轮转
- 监控异常日志

## SQL 注入防护
- 使用参数化查询
- 不要拼接 SQL 语句
- 实施输入验证
- 使用 ORM

## XSS 防护
- 对用户输入进行转义
- 使用 CSP
- 实施输入验证
- 使用安全的模板引擎

## CSRF 防护
- 使用 CSRF 令牌
- 实施 SameSite Cookie
- 验证请求来源
- 使用安全的会话管理
