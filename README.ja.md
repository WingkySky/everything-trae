# Everything Trae

Trae AI エージェント設定コレクション - エージェントのパフォーマンス向上

## 概要

このプロジェクトは、[everything-claude-code](https://github.com/affaan-m/everything-claude-code) プロジェクトの成功経験を参考に、Trae AI プログラミングツールのための完全な設定ソリューションを提供します。専門的なエージェント、スキルモジュール、自動化フック、外部ツール統合（MCP）を設定することで、Trae を通常の AI アシスタントから専門的な開発チームへと変身させます。

## コア機能

### 1. 専門エージェント

6つの専門エージェントを設定しています：

- **planner** - プランナー：要件分析、技術選定、タスク分解
- **coder** - コード生成：コード実装、ドキュメント作成、テスト生成
- **reviewer** - コードレビュー：品質評価、セキュリティチェック、パフォーマンス最適化
- **debugger** - デバッグ専門家：問題の特定、修正案、テスト検証
- **tester** - テスト専門家：テスト戦略、テストケース作成、カバレッジ分析
- **writer** - ドキュメント専門家：APIドキュメント、開発ドキュメント、アーキテクチャドキュメント

### 2. スキルモジュール

4つのコアスキルモジュールを設定しています：

- **tdd-workflow** - テスト駆動開発ワークフロー
- **code-review-standards** - コードレビュー基準とベストプラクティス
- **git-commit-standards** - Gitコミット仕様
- **documentation-standards** - 技術ドキュメント作成基準

### 3. 自動化フック

4つの自動化フックを設定しています：

- **pre-commit** - プリコミットフック：コードチェック、テスト検証、セキュリティスキャン
- **post-commit** - ポストコミットフック：通知、統計、クリーンアップ
- **pre-push** - プリプッシュフック：コードチェック、ビルド検証、コンプライアンスチェック
- **build** - ビルドフック：準備、実行、検証

### 4. 外部ツール統合（MCP）

3つのMCPサーバーを設定しています：

- **filesystem** - ファイルシステム操作
- **git** - Gitバージョン管理
- **memory** - 永続メモリ

## クイックスタート

### インストール

1. `.trae` ディレクトリをプロジェクトのルートディレクトリにコピーします
2. Trae はすべての設定を自動的に読み込みます
3. AI プログラミングアシスタントの使用を開始します

### 使用方法

1. **適切なエージェントを選択**
   ```
   @planner この要件を分析して開発計画を作成してください
   @coder この機能を実装してください
   @reviewer このコードをレビューしてください
   ```

2. **スキルのトリガー**
   - TDD 開発：要件を説明すると、Trae が自動的に TDD ワークフローを使用します
   - コードレビュー：提出前に自動的にレビュープロセスがトリガーされます
   - Git コミット：標準化されたコミットメッセージ形式に従います

3. **MCP ツールの使用**
   - ファイル操作：Trae はプロジェクトファイルの読み書きができます
   - Git 操作：Trae は Git リポジトリの表示と操作ができます
   - メモリ機能：Trae はプロジェクトコンテキストを記憶できます

## ベストプラクティス

### 1. エージェントの使用

- タスクの種類に応じて適切なエージェントを選択
- タスク要件を明確に説明
- 時宜を得たフィードバックと調整を提供

### 2. スキルの使用

- 既存のスキルを再利用
- プロジェクト固有のスキルを作成
- 定期的にスキルを更新

### 3. フックの使用

- 必要なフックを設定
- フック実行時間を最適化
- フック実行状態を監視

### 4. MCP の使用

- 必要な MCP サーバーのみを有効化
- MCP アクセス権限を制限
- 定期的に MCP 設定をレビュー

## everything-claude-code との比較

このプロジェクトは everything-claude-code の設計コンセプトを借用していますが、Trae 向けに最適化されています：

| 機能 | everything-claude-code | Everything Trae |
|------|----------------------|-----------------|
| エージェント | 9つの専門エージェント | 6つのコアエージェント |
| スキル | 11つのスキルモジュール | 4つのコアスキル |
| フック | トリガーベースの自動化 | 4つの主要フック |
| MCP | 外部ツール統合 | 3つのコアMCP |
| 設定 | Claude Code 専用 | Trae 専用 |

## トラブルシューティング

### 一般的な問題

1. **設定が有効にならない**
   - 設定ファイルのパスを確認
   - 設定ファイルの形式を検証
   - Trae ログを確認

2. **パフォーマンスの問題**
   - エージェントの数を減らす
   - スキルのサイズを最適化
   - MCP サーバーの数を減らす

3. **権限の問題**
   - ファイル権限を確認
   - MCP 設定を検証
   - ユーザー権限を確認
