# Everything Trae

Trae AI 에이전트 설정 모음 - 에이전트 성능 향상

## 개요

이 프로젝트는 [everything-claude-code](https://github.com/affaan-m/everything-claude-code) 프로젝트의 성공적인 경험을 참고하여, Trae AI 프로그래밍 도구를 위한 완전한 설정 솔루션을 제공합니다. 전문적인 에이전트, 스킬 모듈, 자동화 후크, 외부 도구 통합(MCP)을 설정하여, Trae를 일반적인 AI 어시스턴트에서 전문적인 개발 팀으로 변화시킵니다.

## 핵심 기능

### 1. 전문 에이전트

6개의 전문 에이전트를 설정했습니다：

- **planner** - 플래너：요구사항 분석, 기술 선택, 작업 분해
- **coder** - 코드 생성：코드 구현, 문서 작성, 테스트 생성
- **reviewer** - 코드 리뷰：품질 평가, 보안 검사, 성능 최적화
- **debugger** - 디버깅 전문가：문제 특정, 수정 방안, 테스트 검증
- **tester** - 테스트 전문가：테스트 전략, 테스트 케이스 작성, 커버리지 분석
- **writer** - 문서 전문가：API 문서, 개발 문서, 아키텍처 문서

### 2. 스킬 모듈

4개의 핵심 스킬 모듈을 설정했습니다：

- **tdd-workflow** - 테스트 주도 개발 워크플로우
- **code-review-standards** - 코드 리뷰 기준 및 모범 사례
- **git-commit-standards** - Git 커밋 사양
- **documentation-standards** - 기술 문서 작성 기준

### 3. 자동화 후크

4개의 자동화 후크를 설정했습니다：

- **pre-commit** - 커밋 전 후크：코드 검사, 테스트 검증, 보안 스캔
- **post-commit** - 커밋 후 후크：알림, 통계, 정리
- **pre-push** - 푸시 전 후크：코드 검사, 빌드 검증, 준수 검사
- **build** - 빌드 후크：준비, 실행, 검증

### 4. 외부 도구 통합（MCP）

3개의 MCP 서버를 설정했습니다：

- **filesystem** - 파일 시스템 작업
- **git** - Git 버전 관리
- **memory** - 영속 메모리

## 빠른 시작

### 설치

1. `.trae` 디렉토리를 프로젝트 루트 디렉토리에 복사하세요
2. Trae가 모든 설정을 자동으로 로드합니다
3. AI 프로그래밍 어시스턴트 사용을 시작하세요

### 사용 방법

1. **적절한 에이전트 선택**
   ```
   @planner 이 요구사항을 분석하고 개발 계획을 작성해주세요
   @coder 이 기능을 구현해주세요
   @reviewer 이 코드를 리뷰해주세요
   ```

2. **스킬 트리거**
   - TDD 개발：요구사항을 설명하면 Trae가 자동으로 TDD 워크플로우를 사용합니다
   - 코드 리뷰：제출 전 자동으로 리뷰 프로세스가 트리거됩니다
   - Git 커밋：표준화된 커밋 메시지 형식을 따릅니다

3. **MCP 도구 사용**
   - 파일 작업：Trae가 프로젝트 파일의 읽기 및 쓰기를 할 수 있습니다
   - Git 작업：Trae가 Git 저장소의 보기 및 조작을 할 수 있습니다
   - 메모리 기능：Trae가 프로젝트 컨텍스트를 기억할 수 있습니다

## everything-claude-code와 비교

이 프로젝트는 everything-claude-code의 설계 개념을 차용했지만, Trae에 최적화되어 있습니다：

| 기능 | everything-claude-code | Everything Trae |
|------|----------------------|-----------------|
| 에이전트 | 9개의 전문 에이전트 | 6개의 핵심 에이전트 |
| 스킬 | 11개의 스킬 모듈 | 4개의 핵심 스킬 |
| 후크 | 트리거 기반 자동화 | 4개의 주요 후크 |
| MCP | 외부 도구 통합 | 3개의 핵심 MCP |
| 설정 | Claude Code 전용 | Trae 전용 |

## 문제 해결

### 일반적인 문제

1. **설정이 적용되지 않음**
   - 설정 파일 경로 확인
   - 설정 파일 형식 검증
   - Trae 로그 확인

2. **성능 문제**
   - 에이전트 수 감소
   - 스킬 크기 최적화
   - MCP 서버 수 감소

3. **권한 문제**
   - 파일 권한 확인
   - MCP 설정 검증
   - 사용자 권한 확인
